# 9. while 문

- 조건이 참(true)인 동안 문장, ...을 반복 실행
- 거짓(False)이 될 때 까지 코드를 반복 실행된다. 거짓이 되면 루프를 빠져나간다.
- 반복 횟수를 명확히 알 수 없는 경우에 주로 사용된다.

## 9.1 while문의 기본구조

```js
while (조건식){
  문장;
  ...
}
```

```js
// 안녕 다섯 번 출력하기
let i = 1;

while (i <= 5) {
  console.log("안녕");
  i++;
}
```

## 9.2 do while 문 기본구조

- 자주 사용하지 않음..

```js
do {
  문장;
  ...
}while(조건식)
```

```js
let x = -10;

do {
  console.log("안녕??");
} while (x > 0);
```
